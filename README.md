# BlackJack-App
A multi-player Blackjack game built with Python and Pygame. Features an event-driven round-robin turn system, object-oriented architecture, and logics to handle player states and deck management

## üÉè Features
* **Scalable Multi-player Support**: The engine is designed to handle multiple player instances within a single game session.
* **Sequential Turn Logic**: A round-robin system manages player actions, automatically handling "Stand" and "Bust" states.
* **Asset Integration**: Dynamically loads card imagery from the local directory to represent the game state.

##  Tech Stack
* **Language**: Python 3
* **Library**: Pygame
